## Что искать в тексте (дисциплина ОВиК)

- Нормативные ссылки -> проверить актуальность по нормативной базе дисциплины
- Расчётные параметры наружного воздуха (температура, энтальпия)
- Расходы приточного/вытяжного воздуха по помещениям
- Тепловые нагрузки на отопление и вентиляцию
- Параметры теплоносителя (температурный график)
- Характеристики вентиляционного оборудования
- Системы дымоудаления и подпора воздуха

## JSON-шаблон project_params

```json
{
  "outdoor_temp_winter_C": -28,
  "outdoor_temp_summer_C": 28,
  "total_supply_air_m3h": 0,
  "total_exhaust_air_m3h": 0,
  "heating_power_kW": 0,
  "coolant_params": "вода 95/70",
  "smoke_exhaust_systems_count": 0,
  "building_height_category": "до 28 м"
}
```

## Описание полей

| Поле | Описание | Пример |
|---|---|---|
| outdoor_temp_winter_C | Расчётная температура наружного воздуха (холодный период) | -28 |
| outdoor_temp_summer_C | Расчётная температура наружного воздуха (тёплый период) | 28 |
| total_supply_air_m3h | Суммарный расход приточного воздуха, м3/ч | 15000 |
| total_exhaust_air_m3h | Суммарный расход вытяжного воздуха, м3/ч | 14500 |
| heating_power_kW | Тепловая мощность системы отопления, кВт | 250 |
| coolant_params | Теплоноситель и температурный график | "вода 95/70", "вода 80/60" |
| smoke_exhaust_systems_count | Количество систем дымоудаления | 3 |
| building_height_category | Категория здания по высоте | "до 28 м", "28-50 м", "свыше 50 м" |
