## 8. ИНСТРУКЦИЯ ПО АКТУАЛИЗАЦИИ
1. Для проверки статуса документа — MCP `mcp__norms__get_norm_status(code)` (authoritative).
2. При ссылке на норму всегда указывай конкретный пункт: `[СП 63.13330.2018, п. 10.3.25]`.
3. Для цитаты пункта — MCP `mcp__norms__get_paragraph_json(code, paragraph)`; без WebSearch.
4. Если нужной нормы нет ни в этом справочнике, ни в Norms-main — фиксируй запрос в `missing_norms_queue.json`, не гадай.
5. В типичных проектах МКД на данном объекте (Alia/Мосфильмовская и аналоги) используются классы бетона В25–В40, арматура A500С (рабочая) и A240 (конструктивная). Отклонения от этого набора требуют обоснования.
