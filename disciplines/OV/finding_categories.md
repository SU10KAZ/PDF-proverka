Категории замечаний для дисциплины ОВиК:

| category | Описание |
|---|---|
| `airflow` | Расходы воздуха: несоответствие расчёту, дисбаланс приток/вытяжка |
| `duct_sizing` | Воздуховоды: сечения, скорости, потери давления |
| `heating` | Отопление: теплопотери, мощность приборов, температурный режим |
| `piping` | Трубопроводы: диаметры, материал, уклоны, арматура |
| `fire_dampers` | Противопожарные клапаны: наличие, расположение, огнестойкость |
| `smoke_exhaust` | Дымоудаление: расход, компенсация, огнестойкость воздуховодов |
| `equipment` | Оборудование: характеристики, подбор, резервирование |
| `insulation` | Изоляция: теплоизоляция трубопроводов и воздуховодов |
| `noise` | Шум: превышение допустимых уровней, шумоглушители |
| `documentation` | Документация: соответствие чертежей спецификации, оформление |
| `normative_refs` | Устаревшие или неверные ссылки на нормативные документы |

Формат в JSON:
```
"category": "airflow|duct_sizing|heating|piping|fire_dampers|smoke_exhaust|equipment|insulation|noise|documentation|normative_refs"
```
