Категории замечаний для дисциплины ЭОМ/ЭС:

| category | Описание |
|---|---|
| `protection` | Защитные аппараты: АВ, УЗО, УЗДП, селективность |
| `cable` | Кабельная продукция: марки, сечения, прокладка |
| `grounding` | Заземление, молниезащита, ОСУП/ДСУП |
| `documentation` | Соответствие чертежей спецификации, оформление |
| `calculation` | Расчётные ошибки: нагрузки, токи, мощности |
| `current_transformer` | Трансформаторы тока: номинал, класс, вторичный ток |
| `phase_balance` | Балансировка фаз, распределение нагрузок |
| `fire_safety` | Пожарная безопасность: ОКЛ, НПО, СПЗ, огнестойкость |
| `normative_refs` | Устаревшие или неверные ссылки на нормативные документы |

Формат в JSON:
```
"category": "protection|cable|grounding|documentation|calculation|current_transformer|phase_balance|fire_safety|normative_refs"
```
