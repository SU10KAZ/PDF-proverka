## Что искать в тексте (дисциплина ЭОМ/ЭС)

- Нормативные ссылки -> проверить актуальность по нормативной базе дисциплины
- Расчётные параметры (Рр, cos_phi, Iр, категория ЭС)
- Марки кабелей и соответствие ГОСТ 31996-2012
- Защитные аппараты из таблиц (номиналы, тип)
- Противоречия между разделами ПЗ

## JSON-шаблон project_params

```json
{
  "power_supply_category": "...",
  "voltage": "0.4 кВ",
  "total_load_kVA": 0,
  "cos_phi": 0,
  "grounding_system": "TN-C-S"
}
```

## Описание полей

| Поле | Описание | Пример |
|---|---|---|
| power_supply_category | Категория надёжности электроснабжения | "I", "II", "III", "I+II" |
| voltage | Напряжение сети | "0.4 кВ", "10 кВ" |
| total_load_kVA | Суммарная расчётная мощность, кВА | 480 |
| cos_phi | Коэффициент мощности | 0.92 |
| grounding_system | Система заземления | "TN-C-S", "TN-S", "TT" |
