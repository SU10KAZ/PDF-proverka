## Задача

Для КАЖДОГО замечания из списка выше:

1. Прочитай текущую формулировку из `03_findings.json`
2. Прочитай результат верификации нормы из `norm_checks.json`
3. Проверь `paragraph_checks` в `norm_checks.json`:
   - Если для замечания есть запись с `paragraph_verified: false` — **цитата нормы некорректна**.
     Используй `actual_quote` для исправления формулировки.
   - Если `paragraph_verified: true` — цитата подтверждена, использовать как есть.
4. Определи:
   - Если норма **отменена** (`cancelled`) — замечание может быть неактуальным.
     Проверь: есть ли замена? Если да — переформулируй со ссылкой на замену.
     Если нет замены — пометь как "требует дополнительной проверки".
   - Если норма **заменена** (`replaced`) — замени ссылку на новый документ.
     Найди аналогичный пункт в новом документе. Если пункт изменился — обнови формулировку.
   - Если **устаревшая редакция** (`outdated_edition`) — обнови номер редакции/изменения.
     Проверь: не изменились ли требования в новой редакции.
   - Если **цитата не подтверждена** (`paragraph_verified: false`) — исправь ссылку на пункт
     и формулировку, используя реальный текст из `actual_quote`.

5. Для каждого замечания запиши:
   - Исходную формулировку (для сравнения)
   - Обновлённую формулировку
   - Что именно изменилось и почему

## Выходной файл

ЗАПИСАТЬ: `{PROJECT_PATH}/_output/03_findings.json`

Прочитай текущий `03_findings.json`, обнови ТОЛЬКО замечания из списка выше, запиши обратно.

Для каждого обновлённого finding добавь поля:
- `"norm_verified": true`
- `"norm_status": "revised"` (или `"warning"` если неопределённо)
- `"norm_revision"`: `{{"original_norm": "...", "revised_norm": "...", "revision_reason": "..."}}`

Для остальных (не в списке): добавь `"norm_verified": true, "norm_status": "ok"`.

## Правила

1. НЕ удаляй и НЕ добавляй замечания — только обновляй существующие
2. Сохрани ВСЕ оригинальные поля каждого finding — добавляй новые, не удаляй старые
3. Пиши JSON через Write — НЕ выводи в чат
4. После записи выведи краткий итог: что изменилось
